		- [Why I Don’t Use Arrow Functions With const/let - YouTube](https://youtu.be/5iGGvJn8K1U)
		  collapsed:: true
			- {{video https://youtu.be/5iGGvJn8K1U}}
				- {{youtube-timestamp 11}} Arrow functions give access to `this` keyword in ways normal functions don't
				- {{youtube-timestamp 61}} They don't have hoisting behaviour, which means they must be defined before code that uses them
					- {{youtube-timestamp 94}} You can put the business logic (e.g. helper functions) BELOW the main code which utilises them if you're not using arrow functions. This keeps the most important info at the top and improves readability
				- {{youtube-timestamp 159}} He uses arrow functions whenever he needs an anonymous function e.g. callbacks
